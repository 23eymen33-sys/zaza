# Geliştirici Rehberi

## Giriş
Bu belge, Zaza uygulamasının geliştirilmesiyle ilgili temel bilgileri ve yönergeleri içermektedir. Amaç, geliştiricilerin projeye hızlı bir şekilde adapte olabilmelerini sağlamaktır.

## Kurulum
### Gereksinimler
- Python 3.8 veya üstü
- Node.js 14 veya üstü

### Kurulum Adımları
1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/23eymen33-sys/zaza.git
   cd zaza
   ```
2. Gereksinimlerinizi yükleyin:
   ```bash
   pip install -r requirements.txt
   npm install
   ```

## Geliştirme
### Kod Düzeni
- Kodunuzun temiz ve okunabilir olmasına dikkat edin.
- Değişiklikler yapmadan önce bir dal oluşturun:
   ```bash
   git checkout -b feature/your-feature-name
   ```

### Testler
Projeyi geliştirdikten sonra testlerinizi çalıştırmayı unutmayın:
```bash
test
```

## Katkıda Bulunma
Katkıda bulunmak istiyorsanız, bir pull request oluşturabilir veya direkt olarak bizimle iletişime geçebilirsiniz.

## İletişim
Sorularınız veya önerileriniz için lütfen repo sahibine ulaşın.
